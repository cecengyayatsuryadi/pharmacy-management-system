# Documentation Index

## Core Docs
- `README.md` -> onboarding cepat, setup, perintah utama.
- `docs/ROADMAP.md` -> arah fase pengembangan produk.
- `docs/CHECKPOINT.md` -> snapshot status teknis sesi terkini.
- `docs/ARCHITECTURE.md` -> prinsip arsitektur dan boundary sistem.
- `docs/RELEASE_AUDIT_CHECKLIST.md` -> gate checklist sebelum release.
- `docs/REPO_REVIEW.md` -> daftar finding teknis aktif.

## Cara Pakai
1. Mulai dari `README.md` untuk setup/run.
2. Lihat `docs/CHECKPOINT.md` untuk status saat ini.
3. Lihat `docs/ROADMAP.md` untuk prioritas berikutnya.
4. Gunakan `docs/REPO_REVIEW.md` + `docs/RELEASE_AUDIT_CHECKLIST.md` sebelum release.
